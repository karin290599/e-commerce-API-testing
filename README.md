
# 🛒 E-commerce Backend API Testing Project

## 📌 Описание проекта

Бэкенд для e-commerce платформы — RESTful API на **Spring Boot** для онлайн-магазина.  
Реализована микросервисная архитектура с использованием Feign-клиентов, JWT-аутентификации, Redis-кэширования профилей и PostgreSQL для хранения данных.

## 🧪 Обязанности и зона ответственности

- Развернула **локальный Docker-стенд** и самостоятельно подготовила тестовое окружение  
- Провела функциональное тестирование REST API  
- Разработала чек-листы и тест-кейсы для модулей системы корзины покупок согласно предоставленной документации: добавление товара в корзину, получение корзины (просмотр содержимого корзины), обновление количества товара в корзине, удаление товара из корзины, очистка корзины.
- Выполнила тестирование  
- Проводила проверку API через инструменты: **Postman, cURL**  
- Работала с базой данных через **DBeaver, cURL** и выполняла SQL-проверки  
- Валидировала ответы API (JSON, коды ошибок, бизнес-логика)  
- Документировала дефекты с указанием приоритета, шагов воспроизведения и фактического результата

В ходе тестирования удалось выявить критические и высокоприоритетные дефекты на ранних этапах разработки.
